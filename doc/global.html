Add new items while removing old items.

Name Type Argument Description
index number The index at which to start changing the array.
howMany number The number of old array items to remove. If howMany is 0, no items are removed. If howMany is greater than the number of items left in the array starting at index or isn't specified, then all of the items through the end of the array will be deleted.
items Object <repeatable>
 The items to add to the array. If you don't specify any items, splice simply removes items from the array.
